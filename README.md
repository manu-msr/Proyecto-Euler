Proyecto Euler
=========================================

Repositorio con la solución a los distintos proyectos del sitio https://projecteuler.net/.

### Repositorio

Para obtener la soluciones se puede clonar el repositorio:

```shell
$ git clone https://github.com/manu-msr/Proyecto-Euler.git
```
